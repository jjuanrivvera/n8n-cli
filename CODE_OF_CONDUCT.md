# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
as its Code of Conduct. The canonical text (including the full standards,
enforcement guidelines, and scope) lives at that link; a translated copy is
available at <https://www.contributor-covenant.org/translations/>.

## In short

- Be respectful, welcoming, and considerate.
- No harassment, discrimination, or personal attacks.
- Assume good faith; critique ideas, not people.

These expectations apply in all project spaces (issues, pull requests,
discussions) and when representing the project in public.

## Reporting

To report unacceptable behavior, contact the maintainer privately via GitHub
([@jjuanrivvera99](https://github.com/jjuanrivvera99)). All reports are reviewed
and handled confidentially, following the Contributor Covenant's enforcement
guidelines.
