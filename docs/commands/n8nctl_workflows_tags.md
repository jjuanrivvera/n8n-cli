## n8nctl workflows tags

Get or replace a workflow's tags

```
n8nctl workflows tags <id> [--set <tagId,...>] [flags]
```

### Options

```
  -h, --help          help for tags
      --set strings   replace tags with these tag ids (comma-separated; empty clears)
```
